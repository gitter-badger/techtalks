# Tech Talks

Host tech events with RSVP, Photo, Notifications, Videos and more.

> A WIP project... most functionality is in development

## Install

```
$ git clone https://github.com/adambutler/tech_talks.git
$ cd tech_talks
$ bundle install
$ rake db:create
$ rake db:migrate
$ cp .env.example .env

## Run Specs

```
# With guard
guard

# .. or without
bundle exec rspec
```
